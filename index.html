Usage examples (copy & paste)
@font-face (recommended):
@font-face {
  font-family: 'ShimulModZUIFont';
  src: url('https://cdn.jsdelivr.net/gh/shimulmodz/Shimul-ModZ-UI-Font/UIFont/ShimulModZUIFont.woff') format('woff');
  font-weight: 100 900;
  font-style: normal;
  font-display: swap;
}

body { 
  font-family: 'ShimulModZUIFont', system-ui, sans-serif; 
}
Preload (place in <head>):
<link rel="preload" 
  href="https://cdn.jsdelivr.net/gh/shimulmodz/Shimul-ModZ-UI-Font/UIFont/ShimulModZUIFont.woff" 
  as="font" 
  type="font/woff" 
  crossorigin>

Generate WOFF2 (recommended) — quick commands

Option A (fonttools):

pip install fonttools brotli
pyftsubset ShimulModZUIFont.ttf --flavor=woff2 --output-file=ShimulModZUIFont.woff2

Option B (Google woff2):

# build the tool then
./woff2_compress ShimulModZUIFont.ttf
# produces ShimulModZUIFont.woff2

Option C (FontForge):

fontforge -lang=ff -c 'Open($1); Generate($2)' ShimulModZUIFont.ttf ShimulModZUIFont.woff

License

I recommend the SIL Open Font License 1.1 (OFL) for fonts. Add LICENSE.txt with the full OFL text in repo root so other devs can use the font freely. Example short note for README:
ShimulModZ UI Font

Licensed under the SIL Open Font License, Version 1.1
See LICENSE.txt for details.

Tag a release for stable jsDelivr URLs

Create a Git tag and push it:
git tag v1.0.0
git push origin v1.0.0

// Then use versioned CDN URL:
https://cdn.jsdelivr.net/gh/shimulmodz/Shimul-ModZ-UI-Font@v1.0.0/UIFont/ShimulModZUIFont.woff

Troubleshooting

  • If font doesn't load: open DevTools > Network — confirm font path returns 200 not 404.
  • Blob links won't work (eg: github.com/.../blob/...). Use raw or jsDelivr links.
  • Ensure file name exactly matches (case-sensitive on CDN).
  • Use crossorigin on preload <link>.
